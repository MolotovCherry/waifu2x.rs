# waifu2x-ncnn-vulkan
waifu2x converter ncnn version, runs fast on intel / amd / nvidia with vulkan

# the original waifu2x project
https://github.com/nagadomi/waifu2x

# Usage
```
waifu2x.exe [input image] [output png] [noise=-1/0/1/2/3] [scale=1/2]
```
